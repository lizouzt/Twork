# Twork-vue-manage

Twork-后台管理

## 项目资源

    - node >= 10.15.1
    - vue-cli 3.0
    - vue 2.6
    - element-ui 2.13.1
    - sass
    - eslint

## 目录结构

    src
        assets/      静态资源
        components/  公用组件
        router/      路由模块
        api/         接口中心
        store/       公共数据缓存
        utils/       公用方法
        config/      全局配置
        views/       基础页面
        views/layout 公用布局

## 正式线打包脚本

    npm run build

## 测试打包脚本

    npm run test
